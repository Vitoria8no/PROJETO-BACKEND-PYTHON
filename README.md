# 🚀 Sistema de Gerenciamento de Usuários (CRUD)

Este é um projeto em *Python* focado em Back-end, onde desenvolvi um sistema modular para gerenciar usuários utilizando um banco de dados *SQLite*.

## 🛠️ Funcionalidades
* *Cadastrar:* Adiciona novos usuários (Nome e Email).
* *Listar:* Exibe todos os usuários cadastrados.
* *Atualizar:* Permite editar o e-mail de um usuário existente.
* *Deletar:* Remove usuários do banco de dados pelo ID.
* *Buscar:* Localiza usuários por nome (ou parte dele).
* *Exportar Relatório:* Gera um arquivo .txt automático com todos os dados.

## 📁 Estrutura do Projeto
* main.py: Interface do usuário e lógica do menu.
* banco.py: Funções de conexão e comandos SQL.
* meu_banco.db: Banco de dados SQLite (gerado automaticamente).
